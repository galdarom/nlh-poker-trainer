# NLH Poker Trainer

A browser-based No-Limit Hold'em trainer. You're dealt a hand at a 7-handed table and
get live 1–10 / 0–100 hand-strength scores (raw, position-adjusted, and after the action
starts), play full streets against four opponent types (LP / LAG / TP / TAG) with stacks,
all-ins and side pots, and can click any opponent to see their range matrix.

**Play it:** open the published page (GitHub Pages) — it's a single self-contained HTML
file, no install or server required.

Built with vanilla JavaScript (no framework, no build step to run).
